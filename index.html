Member Recruiting

新規作業員 募集要項

1. 職種と業務内容

募集職種 ベース（正規作業員）/ ドラム（サポート作業員）
業務内容
  • 既存パートの完全コピーおよび演奏
  • パート制作、レコーディング（ベースのみ）
  • ライブ・リハーサルへの参加

2. 応募資格

必須条件
  • 安定した力強い演奏
  • ツインペダル、楽曲に対応可能な操作スキル(ドラム)
歓迎条件
  • DAW操作/作曲経験(ベース)
  • チャイナシンバルの使用経験(ドラム)

3. 労働条件・待遇

契約期間 2026年7月頃 〜 無期（ベース）/ 2026年9月頃 〜 ドラムメンバー復帰まで
勤務地 高田馬場（スタジオ）、各ライブハウス
待遇
  • 給与：能力考慮のうえ、応相談
  • 制服貸与
  • 譜面支給

4. 応募方法

メールにて下記5点を送付してください。

1. 名前(ニックネーム可能) / 2. 年齢 / 3. 居住エリア / 4. 音楽経歴 / 5. 演奏動画URL

連絡先：ykdband@gmail.com
